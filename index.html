Automação de Formulários Web com Python e RPA

Atuei na implementação de uma rotina de RPA desenvolvida em Python com uso de bibliotecas como pandas, selenium e psycopg2, voltada à automação do preenchimento de formulários em uma aplicação web a partir de dados estruturados em arquivos CSV. A automação resultou em ganho de produtividade, padronização do processo e redução de erros manuais.
